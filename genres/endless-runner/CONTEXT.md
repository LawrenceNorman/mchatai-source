# Endless Runner

An endless runner is a fast-paced game where players navigate a character through an infinite course, avoiding obstacles and collecting items to achieve the highest possible score before inevitably failing.

## When this genre is the right seed

Classifier keywords: endless runner, infinite runner, temple run, subway surfers, obstacle avoidance, high score, pixel art, grid crossing

## Seeded wisdom rules

1. Endless runner games must have simple controls that allow for quick and responsive actions.
2. The game should feature escalating difficulty to maintain player engagement and challenge.
3. A scoring system is essential, rewarding players for distance traveled and items collected.
4. Visual feedback, such as flashes or animations, should be used to indicate milestones or player failure.
5. The game environment should be visually engaging with a retro arcade aesthetic to enhance the experience.
