# Stacker Game

Stacker games challenge players to time their actions precisely to stack moving blocks as high as possible, with increasing difficulty as the stack grows narrower.

## When this genre is the right seed

Classifier keywords: stacker, block stacking, timing, arcade, precision, tower building, alignment bonus, narrowing blocks

## Seeded wisdom rules

1. Blocks must slide horizontally across the screen and require precise timing to drop.
2. Any portion of a block that hangs off the edge should be sliced off, making subsequent blocks narrower.
3. The game ends when a block becomes too narrow to land on the previous one.
4. A perfect stack within a small pixel alignment should grant a bonus and prevent the block from narrowing.
5. The camera should pan up as the tower grows taller to maintain focus on the stacking action.
