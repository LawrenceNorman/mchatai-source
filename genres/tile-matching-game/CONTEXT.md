# Tile Matching Game

Tile matching games involve manipulating tiles on a grid to match and merge them, often using swipe gestures or clicks to combine tiles of the same value, aiming for strategic high scores.

## When this genre is the right seed

Classifier keywords: 2048, threes, tile matching, grid puzzle, merge tiles, swipe controls, strategic planning, minimalist design

## Seeded wisdom rules

1. Tile matching games MUST have a grid layout where tiles can be moved or merged.
2. The game should end when no more moves are possible on the grid.
3. A preview of the next tile can enhance strategic planning.
4. Use smooth animations to enhance the user experience when tiles merge.
5. Design the game with a clean, minimalist aesthetic for mobile devices.
