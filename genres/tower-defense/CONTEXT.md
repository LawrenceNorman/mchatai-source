# Tower Defense

A strategy subgenre where players must defend a base or territory by strategically placing towers that automatically attack waves of enemies. The challenge lies in optimizing tower placement and upgrades to withstand increasingly difficult enemy waves.

## When this genre is the right seed

Classifier keywords: tower defense, strategy, base defense, enemy waves, tower placement, tower upgrades, path management, resource management

## Seeded wisdom rules

1. Tower defense games MUST have a clear path for enemies to follow, which players can influence through tower placement.
2. Players should have the ability to upgrade towers to increase their effectiveness against stronger waves.
3. Resource management is crucial; players must balance spending on new towers versus upgrading existing ones.
4. The game should progressively increase in difficulty, introducing new enemy types and challenges.
5. Players should be able to see the impact of their strategic decisions in real-time as waves progress.
