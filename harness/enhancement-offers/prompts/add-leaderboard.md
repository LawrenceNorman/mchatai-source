Add a public leaderboard to this game using the web-components addon recipe `recipe.with-leaderboard`{{recipeNote}}. Import `GameOverWithRank` from `../../ui/GameOverWithRank.js` and `Leaderboard` from `../../ui/Leaderboard.js`. ON game-over (and only on game-over), call `await GameOverWithRank.show({ host, score: finalScore, metadata: { level, moves }, onRestart: () => game.restart() })`. Add `recipe.with-leaderboard` to the marker `addons` array, and include both `ui.leaderboard` and `ui.game-over-with-rank` in the marker `components` array. Preserve ALL existing game behavior — this is purely additive. The board-id is supplied by the host page bridge; the generated game never sets it. When the player isn't signed in, GameOverWithRank automatically renders a 'Sign in at mchatai.com to post your score' CTA — do not suppress or replace that.
