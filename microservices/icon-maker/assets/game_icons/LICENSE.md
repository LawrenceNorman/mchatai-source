# Game Icons — Provenance & License

The PNGs in `png/` are derived from SVGs downloaded from
[game-icons.net](https://game-icons.net) (full transparent SVG archive,
retrieved 2026-07-02). Modifications: rasterized to 1024×1024,
recolored black→white preserving alpha, and composited onto generated
backgrounds at render time by the icon-maker compositor.

Per-icon attribution (author + source URL) lives in `LEDGER.json`.
The `/credits` web page on mchatai.com is generated from that ledger.

License: [CC BY 3.0](https://creativecommons.org/licenses/by/3.0/)
(two contributors — Viscious Speed and Zeromancer — publish under CC0,
as noted in the upstream license text below).
