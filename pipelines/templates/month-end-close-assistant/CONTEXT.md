# Month-End Close Assistant

CSV-first finance operations workflow for pasted/exported close data. It previews the CSV with `builtin.csvTable`, then drafts a close readiness report with exceptions, reconciliation checks, a checklist, data-quality notes, missing data, and a human-review disclaimer.

Expected inputs include trial balance exports, bank activity exports, invoice/payment exports, or a combined close worksheet. Private finance data should stay `confidential`; do not add public web search to this template without an explicit user request.
